An H\+T\+TP request contains information relative to a received H\+T\+TP frame such as its method, header fields and content. Once H\+T\+TP requests should be processed through a router function. During routing step it is up to the user to forge response data and forward it to the http request instance to be sent back over to the sender of the request.
