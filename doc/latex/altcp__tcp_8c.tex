\doxysubsection{Detailed Description}
Application layered T\+CP connection A\+PI (to be used from T\+C\+P\+IP thread)~\newline
This interface mimics the tcp callback A\+PI to the application while preventing direct linking (much like virtual functions). This way, an application can make use of other application layer protocols on top of T\+CP without knowing the details (e.\+g. T\+LS, proxy connection).

This file contains the base implementation calling into tcp.
